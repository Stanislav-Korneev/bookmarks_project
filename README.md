Приложение для создания закладок.
- Адаптивный дизайн.
- Полная доступность с клавиатуры.
- Фавиконки закладок автоматически подгружаются из google.
- Есть верификация(Необходимо заполнить поля name + url).
- При успешных действиях появляется соответствующая нотификация.


Основной функционал: 
- добавление,
- редактирование,
- удаление,
- копирование url.

Сортировка закладок по:
- алфавиту,
- дате создания,
- активности,
- поисковому запросу.
